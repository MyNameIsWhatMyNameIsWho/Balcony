## Audio folder conventions (recommended)

Put your audio files under `res://assets/audio/` so you can drag-drop them into `data/intro_sequence.tres`.

Suggested structure:

- `assets/audio/foley/`  (close/clean “human scale” sounds)
  - door_open, door_close
  - keys_ring
  - shoes_off
  - footsteps_(wood|tile|carpet)
  - exhale
- `assets/audio/ambience/` (room tone, outside noise, distant city)
- `assets/audio/vo/` (voice-over files later, ideally one per intro block)

Formats:
- Use `.wav` for short SFX you may edit a lot.
- Use `.ogg` for longer ambience/VO to keep size down.
